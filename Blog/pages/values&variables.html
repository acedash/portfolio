Values and Variables

Values:- is a piece of data which we store in a variable.

Variables:-is where store diffrent types of data.In real world variable is like bucket or box in which we store something.

Conventions of Naming Variables.

  1. 1.Variable name cannot not start with number.
  2. 2.We should not start variable name with uppercase letter.
  3. 3.Only $ and _ are only allowed to use to start variable name with.
  4. 4.Variable name can only contain number, letters , underscores ,dollar sign.
  5. 5.Symbols and some reserved words that cannot be used:- &, new, function, name.
  6. 6.Variable names that are all in uppercase are reserved for CONSTANTS that we know will never change eg PI.
