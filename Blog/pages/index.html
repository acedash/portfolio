Javascript Introduction

Javascript is a Hight Level,Object Oriented, Multi-Paradigm programming language.

Hight Level:-Where we dont have to worry about complex stuff like memory mangement

Object Oriented:-Language is mostly based on the concepts like objects for storing most kinds of data.

Multi-Paradigm:-Language is flexible and versatile here we can use all kinds of diffrent programming styles such as imperative and declarative programming and these diffrent styles are diffrent ways of structuring our code.

Role of Javascript in Web Devlopment

HTML:- is responsible for the content of web page eg:- BUTTONS , LINKS, TEXT,IMAGES ETC

CSS:- is responsible for styling of web page or how the content should appear on web page example adding colors, Positioning,layoutn etc

JAVASCRIPT:- is a programming language which helps to add dynamic features on web pages

To understand more easily consider HTML as NOUN , CSS as ADJECTIVE & JS as VERB

Where to Write JAVASCRIPT code

  • 1. Internal Js
  • 2. External Js

Internal Javascript

We can write js in the html file inside SCRIPT tag.You can add script tag in the head or anywhere in the body tag.it is better write SCRIPT tag in the bottom of body tag.

Example

<html>
<head>
<title>Home</title>
<body>
<p>hello world!</p>
<script>
console.log("hello javascript")
</script>
</body>

External Javascript

External javascript is writing code in seprate js file.Which makes code clean.To link js file we use SCRIPT tag.

Example

<html>
<head>
<title>Home</title>
<body>
<p>hello world!</p>
<script src="script.js"></script>
</body>
